# 3.2 Replica Implementation Plan

## Context

Read before starting:
- `docs/2.2_replica.md` — full spec (critical notes on async Redis, lifespan, wake cursor, A-S model, tests)
- `docs/2.1_core.md` — core data contract (what the replica consumes)
- `docs/1_high_level.md` — system overview, canonical payloads

Key gotchas to keep in mind throughout:
1. **Async Redis only** — `import redis.asyncio as redis`, never sync client
2. **Lifespan context manager** — not deprecated `@app.on_event`
3. **Wake cursor = `"$"`** — skip history on activation, only new events
4. **`block=1000` not `block=0`** — use 1-second timeout so the consumer loop can
   evaluate `is_active` between reads (block=0 hangs indefinitely, ignoring sleep commands)
5. **CORS middleware required** — UI on port 3000 POSTs to replicas on 8001/8002,
   browsers block cross-origin without it

## Step 1: Config + Engine (A-S model + local state) ✅ DONE

Create `replica/config.py` and `replica/engine.py`.

**config.py**: All tunables from env with defaults — REPLICA_ID, HOST, PORT, REDIS_URL,
STREAM_KEY, A-S params (GAMMA, K, TICK_SIZE, LEVEL_SIZES), MATCH_DURATION.

**engine.py**: The `ReplicaEngine` class:
- Local state fields: ball_x, ball_y, score, half, clock, vol, mid, book
- `update(event_dict)`:
  1. Apply event fields to local state
  2. Compute reservation price: `r = mid - q * gamma * sigma^2 * (T - t)`
     (q=0 for symmetric book)
  3. Compute optimal spread: `delta = gamma * sigma^2 * (T-t) + (2/gamma) * ln(1 + gamma/k)`
  4. Best bid = `r - delta/2`, best ask = `r + delta/2`
  5. Generate 4 more levels at TICK_SIZE increments with LEVEL_SIZES quantities
  6. Store as `self.book = {"bids": [...], "asks": [...]}`
- `get_world_state(replica_id, lag_ms)` -> returns the full WorldState dict
  matching the Replica -> UI data contract

## Step 2: Publisher + Main (WS broadcast + FastAPI app + consumer) ✅ DONE

Create `replica/publisher.py`, `replica/main.py`, and `replica/requirements.txt`.

**publisher.py**: `Broadcaster` class:
- `self.clients: set[WebSocket]`
- `connect(ws)` / `disconnect(ws)`
- `async broadcast(world_state_dict)`: JSON serialize, send to all clients,
  catch and drop dead connections

**main.py**: Everything network-facing:
- Module-level state: `is_active = False`, `injected_lag_ms = 0`, `last_id = "$"`
- `ReplicaEngine` and `Broadcaster` instances
- Async Redis client: `redis.asyncio.Redis.from_url(config.REDIS_URL, decode_responses=True)`

- **CORS middleware** (required — UI on :3000 POSTs to replica on :8001/:8002):
  ```python
  from fastapi.middleware.cors import CORSMiddleware
  app.add_middleware(CORSMiddleware, allow_origins=["*"], allow_methods=["*"], allow_headers=["*"])
  ```

- **Lifespan**:
  ```python
  @asynccontextmanager
  async def lifespan(app):
      task = asyncio.create_task(consume_redis_stream())
      yield
      task.cancel()
  ```

- **Consumer loop** (`consume_redis_stream`):
  - While True: if not `is_active`, `await asyncio.sleep(1)`, continue
  - **`await r.xread({"game:events": last_id}, block=1000)`** (1-second timeout, NOT block=0)
    - `block=0` blocks indefinitely — if the UI sends `POST /control {"action": "sleep"}`
      while blocked, the loop won't wake up to evaluate `is_active = False` until a new
      Redis event arrives. With `block=1000`, xread returns `None` after 1 second, the
      loop re-evaluates the flag, and can cleanly go back to idle.
  - If result is `None` (timeout), continue loop
  - Parse event, `engine.update(event)`, compute lag_ms
  - If `injected_lag_ms > 0`: `await asyncio.sleep(injected_lag_ms / 1000)`
  - `await broadcaster.broadcast(engine.get_world_state(...))`
  - Update `last_id` to consumed stream ID
  - Wrap in try/except for Redis reconnect

- **Routes**:
  - `GET /api/state` -> `engine.get_world_state(...)`
  - `GET /api/health` -> lag, uptime, event count
  - `POST /api/control` -> set `is_active`, reset `last_id = "$"` on wake
  - `POST /api/chaos` -> set `injected_lag_ms`

- **WebSocket endpoint** `WS /ws`:
  - `broadcaster.connect(ws)`, loop receiving (keepalive), `broadcaster.disconnect(ws)` on close

**requirements.txt**: `fastapi>=0.110.0`, `uvicorn[standard]>=0.29.0`, `redis>=5.0.0`, `websockets>=12.0`
